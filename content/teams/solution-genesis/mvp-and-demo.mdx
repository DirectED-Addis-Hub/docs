# Solution Genesis

## Problem
Skilled Ethiopian freelancers can secure international work, but getting paid is unreliable (PayPal/Stripe unavailable, transfers slow/costly, risky intermediaries).

## MVP
### Scope
- Milestone-based escrow payments
- Lock funds and release per milestone
- Dispute/refund path without intermediaries

### Cardano Integration
Cardano smart contracts provide escrow + milestone release and transparent settlement.
